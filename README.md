# HH Resume Rise Bot 🚀

Telegram-бот, который **сам поднимает ваше резюме на hh.ru каждые 4 часа 10 минут**.

Забудьте про рутину: зашёл раз, ввёл код из письма — и бот дальше работает 24/7.
Просмотры растут, резюме в топе поиска, вы отдыхаете.

### Что делает бот

- Авторизуется по коду из письма (один раз или если сессия протухла)
- Сохраняет сессию в `hh-auth.json` (работает до 24 часов)
- Каждые **4 часа 10 минут** заходит → кликает «Поднять в поиске»
- Присылает в чат красивый отчёт:
  - Название резюме
  - Когда последний раз поднималось
  - Сколько просмотров
  - Сколько приглашений
- Если сессия слетела — сам просит новый код
- Логирует всё в консоль + присылает ошибки в Telegram

### Быстрый старт (3 минуты)

## 1. Склонируй или скачай проект
```bash
git clone
cd hh-resume-rise-bot```

## 2. Установи зависимости
```bash
npm install
```

## 3. Создай файл .env в корне и заполни:env
```bash
BOT_TOKEN=123456:ABC-DEF1234ghIkl-zyx57W2v1u123ew11
RESUME_URL=https://hh.ru/applicant/resumes/ваш-id-резюме
HH_LOGIN=ваша@почта.ru
```

## 4. Запусти
```bash
node src/index.js
```

## 5. Напиши боту /start → введи код из письма один раз → готово!
